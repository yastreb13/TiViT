
# TiViT
В начале семестра была поставлена Стахановская задача написать абсолютно все лабораторные работы по физике (теорию к ним) за неделю. Во время выполнения я понял, что сил на это уходит немало. В этот момент мне пришла идея.

С итальянского десерт *tiramisù* буквально переводится как **«взбодри меня»** или **«подними мне настроение»**. Я решил объединить эту концепцию с Computer Vision. Когда модель видит, что я устал, то предлагает мне заказать тирамису из известных магазинов доставок.

В процессе выполнения для обучения я хотел использовать Driver Drowsiness Dataset (DDD), но он плохо определял усталость при моих условиях. Пришлось использовать одногруппников для составления собственного. По 100 фото уставших и бодрых лиц, 1200 фото, сделанных благодаря 6 добровольцам.

*Веса обученной модели не выкладываю*, так как обучал на небольшом количестве эпох, я думаю, вы сможете обучить лучше
*Датасетом делиться тоже не буду* — одногруппники согласились фоткаться только с условием сохранности данных.

---

**TiViT (Tiramisu Vision Transformer)** — это пет-проект на базе архитектуры Vision Transformer (ViT), который в реальном времени следит за моим лицом через веб-камеру.

- **PyTorch**: Fine-Tuning предобученной модели ViT-B/16  
- **OpenCV**: Захват видеопотока с веб-камеры и детекция лица  
- **Python (webbrowser)**: Автооткрытие сайта-доставки
